- ✅ You@YourDomain.com — any name, any domain
- ✅ Live delivery — emails appear instantly
- ✅ Image upload — profile/company logo
- ✅ SMTP support — send to real email addresses
- ✅ Works on all devices — phone, tablet, desktop

## Setup SMTP
After deploying, add these environment variables:
- `SMTP_HOST` = smtp.gmail.com
- `SMTP_PORT` = 587
- `SMTP_USER` = your-email@gmail.com
- `SMTP_PASS` = your-app-password
